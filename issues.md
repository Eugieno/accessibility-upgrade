## Issues

### 1 – Semantic elements
#### User story 
As a marketing agency
I WANT a codebase that is optimised for semantic elements
SO THAT our site is accessible to ALL users

#### Acceptance Criteria 
Semantic HTML elements can be found throughout the source code

SO THAT our site is accessible to ALL users

#### Acceptance criteria

SO THAT our site is accessible to ALL users

#### Acceptance criteria
